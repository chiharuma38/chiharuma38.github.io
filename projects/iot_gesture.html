IoT Gesture Recognition System

IoT fertilizer spiller with ESP32, dual-mode operation (manual and gesture), and a neural network deployed on-device for real-time agricultural automation.

ESP32 Machine Learning IoT TinyML Embedded Systems

Project Overview

Designed and built an automated fertilizer dispensing system using an ESP32 microcontroller. The device supports two modes: manual control via button input, and gesture-based control using an IMU sensor and an on-device neural network.

  • Gesture Recognition: Trained a TinyML model to classify hand gestures in real time on the ESP32 — no cloud connection required.
  • Dual-Mode Operation: Seamless switch between manual and automatic gesture-driven dispensing.
  • Agricultural Application: Designed for precision agriculture — reduces fertilizer waste through targeted, gesture-triggered dispensing.

Technical Highlights

  • ESP32 microcontroller with integrated Wi-Fi and Bluetooth for IoT connectivity.
  • IMU sensor for gesture input — accelerometer and gyroscope data sampled in real time.
  • TensorFlow Lite model quantized and deployed directly on the ESP32 using TinyML techniques.
  • Python-based training pipeline using collected gesture data with SVM and neural network classifiers.

Lessons Learned

  • On-device inference is feasible on constrained hardware when models are properly quantized.
  • Gesture datasets require careful collection across multiple users to avoid overfitting to one person's movement style.
  • Dual-mode fallback (manual override) is essential for reliable real-world deployment.
