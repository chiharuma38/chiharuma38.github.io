Automatic Medication Dispenser

Arduino-based automatic medication dispensing system with scheduled alerts, safety lock, ultrasonic sensing, and LED notification — designed for elderly and chronic care patients.

Arduino Embedded Systems Healthcare Sensors C Programming

Project Overview

Designed and built an automatic medication dispenser to help elderly and chronic illness patients maintain medication schedules. The system provides timed dispensing, missed-dose alerts, and a safety lock to prevent accidental double-dosing.

  • Scheduled Dispensing: RTC (real-time clock) module triggers dispensing at pre-programmed times — configurable via serial interface.
  • Safety Lock: Servo-controlled door mechanism prevents access between scheduled dispensing windows.
  • Presence Detection: Ultrasonic sensor confirms medication has been removed from the tray after dispensing.
  • Missed Dose Alert: LED and buzzer alert if medication is not taken within a set window after dispensing.

Technical Highlights

  • Arduino Uno with DS3231 RTC for accurate timekeeping independent of power interruptions.
  • HC-SR04 ultrasonic sensor for non-contact medication removal detection.
  • Servo motor controlling the dispensing gate, driven by Arduino PWM output.
  • 16x2 LCD display showing current time, next scheduled dose, and system status.

Lessons Learned

  • Healthcare devices require fail-safe design — the system defaults to locked on power loss to prevent accidental access.
  • User feedback (LCD, LED, buzzer) is essential for devices used by non-technical users.
  • RTC drift over time needs periodic calibration for long-term deployment reliability.
